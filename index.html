1
Créer une clé API Bybit
  • Allez dans Bybit → Account → API Management
  • Créez une clé avec permissions Trade uniquement (pas de retrait !)
  • Notez la clé et le secret
2
Créer un fichier .env
⚠️ Ce fichier ne doit JAMAIS être pushé sur GitHub. Il est déjà dans le .gitignore.
bash 
BYBIT_API_KEY=votre_cle_ici
BYBIT_API_SECRET=votre_secret_ici
INITIAL_BALANCE=200.0
LIVE_TRADING=true
3
Modifier config.py
python 
import os
from dotenv import load_dotenv
load_dotenv()

LIVE_TRADING     = os.getenv('LIVE_TRADING', 'false') == 'true'
BYBIT_API_KEY    = os.getenv('BYBIT_API_KEY', '')
BYBIT_API_SECRET = os.getenv('BYBIT_API_SECRET', '')
4
Modifier market_data.py
python 
exchange = ccxt.bybit({
    'apiKey':          BYBIT_API_KEY,
    'secret':          BYBIT_API_SECRET,
    'enableRateLimit': True,
})
5
Activer les vrais ordres dans paper_broker.py
Dans open_position(), si LIVE_TRADING est True, remplacez la simulation par :
python 
if LIVE_TRADING:
    exchange.create_order(
        symbol, 'market', 'buy' if direction=='long' else 'sell',
        token_size
    )

FONCTIONNALITÉS AVANCÉES
🎯
Trailing Stop
Sécurise les gains dès +1.5% de profit. Le SL suit le prix automatiquement, jamais en arrière.
❄️
Cooldown SL
4h de pause sur une paire après un Stop Loss. Évite de re-rentrer immédiatement sur un mauvais setup.
📊
Filtre ATR
N'entre que si la volatilité dépasse 0.8% / bougie. Élimine les marchés trop calmes pour atteindre le TP.
📈
Filtre MA200
Ne trade qu'en direction de la tendance longue. Long au-dessus de la MA200, Short en dessous.
✋
Contrôle Manuel
Fermeture forcée ou ajout de mise (DCA) directement depuis le dashboard en un clic.
🧠
Moteur Adaptatif
Après 10 trades par paire, ajuste automatiquement RSI, SL et TP selon les performances récentes.
